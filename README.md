## 📖 Overview

YOURTUBE is a full-stack YouTube-inspired video streaming platform developed as part of the **Elevance Web Development Internship**.

The application recreates the core YouTube experience while integrating modern web development practices such as secure authentication, RESTful APIs, responsive design, modular architecture, video management, user engagement features, and scalable backend services.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/ROSHAN-KUMAR-DUTTA/YOURTUBE-ELEVANCE-INTERNSHIP-PROJECT.git
```

---

## Install Dependencies

### Frontend

```bash
cd yourtube
npm install
```

### Backend

```bash
cd ../server
npm install
```

---

# ▶ Run Project

### Backend

```bash
npm start
```

### Frontend

```bash
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=

MONGODB_URI=

JWT_SECRET=

EMAIL_USER=

EMAIL_PASS=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
